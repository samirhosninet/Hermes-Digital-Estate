Install Digital State Stack

This installer checks and installs the three parts that work together: Hermes Agent, Hermes Workspace, and the Digital State profile. Failures stay visible with a clear log and a manual fallback command.

Order matters: install Hermes Agent first. The official Hermes Windows installer provisions Python, Node.js 22, and PortableGit. Then install Hermes Workspace and Digital State. The final step starts Hermes Gateway and the Workspace UI.
